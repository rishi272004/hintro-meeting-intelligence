# Changelog

## [1.0.0] - 2026-06-10

### Added
- Express + TypeScript project structure
- Prisma schema for users, meetings, transcripts, analyses, action items, and reminder logs
- JWT authentication
- Meeting creation, retrieval, and listing
- AI-powered transcript analysis with grounded citations
- Action item creation, status updates, and filtering
- Overdue action item detection
- Hourly reminder scheduler
- Resend email integration for reminders
- Unified response format and trace IDs
- Validation, logging, and centralized error handling
- Swagger documentation
- Mocked unit tests for the main API flows
